# Personal Portfolio Website

## 📌 Project Overview

This is a responsive and interactive Personal Portfolio Website developed as part of my internship at Researcher Lyceum Private Limited. The portfolio showcases my skills, projects, achievements, and contact information in a professional and user-friendly manner.

The website includes advanced features such as an AI-powered chatbot, EmailJS contact form, project modals, testimonials, image gallery with lightbox functionality, and modern UI/UX enhancements.
